# dotfiles
Various dotfiles for system configuration

Currently included are configuration files for Vim and TMux, as well as the tmuxp session directory.

## Configuration
* vimrc -> ~/.vimrc
* tmux.conf -> ~/.tmux.conf
* tmuxp/ -> ~/.tmuxp
